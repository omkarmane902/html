1)Difference between block level element and inline element




Block Level Element Inline Level Element
1) Block Level Element occupies 100% width of its parent element 1) Inline Level Element the occupies content width
2) Block Level Element start with the new line 2) Inline Level Element start with the same line
3) We can give width and hight to block level element 3) We can not give width and hight to Inline level element
4) div , p tag , h1 to h6 tag , footer , nav etc 4) span tag , img tag , almost formating tag





2)Difference between div tag and span tag

Div tag Span tag
1) Div tag occupies 100% width of its parent element 1) Span tag the occupies content width
2) Div tag start with the new line 2) Span start with the same line
3) We can give width and hight to div tag 3) We can not give width and hight to Span tag
4) div , p tag , h1 to h6 tag , footer , nav etc 4) span tag , img tag , almost formating tag





3)Difference between pair tag and unpaired tags

Pair tag Unpaired tag
1) It has opening as well as closing tag 1) It has only the opening tag which act as a self closing tag
2) It is also called as Container tag 2) It is also called as void ,non-container self closing
3) p tag , div tag 3) br tag , hr tag ,input tag
